# 🚀 Simple Craft Launcher

Простой и быстрый лаунчер для Minecraft. Создан с упором на минимализм и легкость.

## 📋 О проекте
**Simple Craft Launcher** — это удобная оболочка для запуска игры. Никакой лишней рекламы или тяжелых скриптов.

## 🛠 Возможности
* **Легкий интерфейс:** Написан на CustomTkinter.
* **Быстрый запуск:** Минимум времени на загрузку игры.
* 
## 📦 Как начать работу
1. Скачайте последнюю версию лаунчера из раздела **Releases**.
2. Поместите `.exe` файл в удобную для вас папку.
3. Запустите его и наслаждайтесь игрой!

## ⚙️ Технические требования
* Windows 10/11
* Установленная Java (для запуска Minecraft)
